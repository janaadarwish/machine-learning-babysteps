# 📌 Internship Tasks

## ✅ TASK 1: Iris Flower Classification

### 📖 Project Description
This project builds a machine learning classification model to identify Iris flower species:
- Setosa
- Versicolor
- Virginica

The classification is based on flower measurements such as:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

### ⚙️ Workflow
- Data loading
- Exploratory Data Analysis (EDA)
- Data preprocessing
- Train-test split
- Model training (Random Forest / Logistic Regression)
- Model evaluation (Accuracy, Confusion Matrix)

### 🎯 Outcome
- Achieved high classification accuracy
- Understood core concepts of supervised learning and classification

## ✅ TASK 2: Unemployment Analysis with Python

### 📖 Project Description
This project analyzes unemployment rate data representing the percentage of unemployed individuals in India.

### 🎯 Objectives
- Clean and preprocess unemployment dataset
- Explore unemployment trends over time
- Investigate the impact of COVID-19
- Identify seasonal patterns
- Generate insights for economic and social policy understanding

### 🛠️ Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

### ⚙️ Workflow
- Data cleaning
- Date formatting and feature extraction
- GroupBy operations for trend analysis
- Time series visualization
- COVID-19 period comparison

### 📊 Key Insights
- Unemployment peaked during COVID-19 lockdown period
- Seasonal fluctuations observed across months
- Significant regional variations
