# 🤖 Machine Learning Baby Steps

A beginner-friendly collection of Jupyter notebooks covering foundational machine learning concepts — built for anyone starting their ML journey from scratch.

---

## 📚 What's Inside

This repo covers three core areas of machine learning:

### 🧹 Data Preprocessing & EDA
- Exploring and understanding datasets
- Handling missing values and outliers
- Feature scaling and encoding
- Visualizing data distributions and correlations

### 🎯 Supervised Learning
- Learning from labeled data
- Regression and classification algorithms
- Model evaluation and performance metrics
- Train/test splits and cross-validation

### 🔍 Unsupervised Learning
- Finding patterns in unlabeled data
- Clustering algorithms (e.g., K-Means)
- Dimensionality reduction
- Evaluating unsupervised models

---

## 🛠️ Requirements

Make sure you have the following installed:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

---

## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/janaadarwish/machine-learning-babysteps.git
   cd machine-learning-babysteps
   ```

2. Launch Jupyter:
   ```bash
   jupyter notebook
   ```

3. Open any notebook and start learning! 🎉
